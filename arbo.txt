javastone
	- javastone.jar
	- crea
		- un fichier par creature
			- Nom du fichier : nom de la creature.crea
			
			Nom			(str)
			atk			(int+)
			vie			(int+)
			guardien 	(0/1)
	
	- carte
		- un fichier par create
			- Nom du fichier : nom de la carte.carte
			
			Nom			(str)
			descr		(str)
			eff			(str, metacode)
			mana		(int+)
			max			(int+)
			
			
	- deck
		- un ficheir par deck
			- Nom du fichier : nom du deck.deck
			
			Nom carte1
				...
			Nom carteN
			
			
Effects:
	TYPE	(int : INVOKE = 0, BUFF = 1, DAMAGE = 2, DECK = 3)
	QUI		(int : 0(tous), 1(joueur), 2(adversaire))
	ARGS:
		INVOKE :
			Nom creature	(str) (nom du fichier sans extension)
		BUFF :
			life			(int)
			dmg				(int)
			
		DAMAGE:
			dmg				(int)
			
		DECK:
			nbr				(int)
			
	Ex:
		Vous piocher 2 carte : 3 1 2
		Invoque Creatur : 0 1 Creatur
		Invoque Castor Garou et pioche une carte : 0 1 Castor_Garou; 3 1 1
